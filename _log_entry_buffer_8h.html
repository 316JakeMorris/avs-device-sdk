◆ ACSDK_LOG_ENTRY_BUFFER_SMALL_BUFFER_SIZE

#define ACSDK_LOG_ENTRY_BUFFER_SMALL_BUFFER_SIZE   128

The size of LogEntryBuffer::m_smallBuffer. Instances of LogEntryBuffer are expected to be allocated on the stack in most use cases. Rather than pick a value that would be large enough for almost any normal log lines (e.g. 4096), a smaller value (128) that will handle the vast majority of typical log lines was chosen to reduce the impact on the stack.

#ifndef used here to allow overriding this value from the compiler command line.
