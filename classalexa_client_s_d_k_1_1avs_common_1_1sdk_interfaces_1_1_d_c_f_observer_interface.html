Detailed Description

This interface is used to observe changes to the state of the DCF delegate.

)
pure virtual

Notification that an DCF delegate state has changed.

Note
Implementations of this method must not call DCFDelegate methods because the DCFDelegate may be in a 'locked' state at the time this call is made. If you do, then you may end up with a deadlock.
Parameters
newStateThe new state of the DCF delegate.
errorThe error associated to the state change.
